# React Product Search & Pagination App

A React.js project built with Vite that displays products from DummyJSON API with search, pagination, and loading functionality.

## Features

- Fetch products using Fetch API
- Search products by:
  - Product Title
  - Product Category

- Client-side Pagination
- Loading State
- Conditional Rendering
- Responsive Product Cards
- Dynamic Search Filtering
- Product Details Display

---

## Tech Stack

- React.js 19
- JavaScript (ES6+)
- Vite

- Fetch API

## Installation

Clone repository:

```bash
git clone <your-repository-url>
```

Move into project:

```bash
cd your-project-folder
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```

Application runs on:

```bash
http://localhost:5173
```
